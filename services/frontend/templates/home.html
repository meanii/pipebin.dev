pipebin.dev

Minimal pastebin for developers. Pipe logs, code, and stdout — instantly.

Quick start

$ dmesg | curl -X POST https://pipebin.dev --data-binary @-
→ https://pipebin.dev/p/xk9d2m

$ cat deploy.log | curl https://pipebin.dev --data-binary @-
→ https://pipebin.dev/p/r4n8pq

$ curl https://pipebin.dev -d "$(git diff HEAD~1)"
→ https://pipebin.dev/p/m2k5vx

Options

# Set expiry (1h, 6h, 24h, 7d, 30d)
$ curl -F "expires=1h" -F "c=@logfile.txt" https://pipebin.dev

# Set a title
$ curl -F "title=deploy-log" -F "c=@deploy.log" https://pipebin.dev

# Retrieve later (plain text, always)
$ curl https://pipebin.dev/raw/xk9d2m

API

Method Endpoint Description
POST / Create a new paste
GET /p/{id} View paste — HTML in browser, plain text via curl
GET /raw/{id} Raw plain text, always

Features

  • No login required
  • Large log support
  • Raw plain text output
  • Expiring pastes
  • curl-friendly API
  • Browser + curl aware
  • Fast and minimal
  • No JavaScript required
