Quick start

# pipe anything — preserves whitespace and indentation
$ cat file.go | curl -sT - {{ .BaseURL }}/
→ {{ .BaseURL }}/p/xk9d2m

# with metadata  (?t=title  ?lang=go  ?e=24h  ?once=1)
$ dmesg | curl -sT - "{{ .BaseURL }}/?t=dmesg&lang=bash&e=6h"
→ {{ .BaseURL }}/p/ab3f7x

# burn after reading — deleted on first view
$ echo "secret token" | curl -sT - "{{ .BaseURL }}/?once=1"
→ {{ .BaseURL }}/p/zz9k1q
   ⚠  burn after reading — deleted on first view

# fetch raw output
$ curl -s {{ .BaseURL }}/raw/xk9d2m

# JSON body also works
$ curl -s {{ .BaseURL }}/ \
    -H 'Content-Type: application/json' \
    -d '{"title":"hello","content":"world","language":"go"}'
→ {{ .BaseURL }}/p/cd4e8w

Shell alias

# Add to ~/.bashrc or ~/.zshrc
pb() { curl -sT - "{{ .BaseURL }}/$@"; }

# Usage
$ cat crash.log | pb
→ {{ .BaseURL }}/p/xk9d2m

$ cat secret.txt | pb '?once=1'
→ {{ .BaseURL }}/p/zz9k1q
   ⚠  burn after reading — deleted on first view

$ cat deploy.sh | pb '?t=deploy&lang=bash&e=24h'
→ {{ .BaseURL }}/p/ab3f7x

API

Method Endpoint Description
POST PUT / Create paste — raw body, JSON, or form
GET /p/{id} View paste in browser
GET /raw/{id} Raw plain text

Query params for raw pipe: ?t=title ?lang=go ?e=24h or ?expires=24h ?once=1 burn after reading

Features

  • No login required
  • Pipe-friendly — raw, JSON, or form
  • Syntax highlighting with line numbers
  • Language auto-detection
  • Burn after reading (?once=1)
  • Expiring pastes (?e=24h)
  • Download button
  • curl-friendly plain text URL
  • Shell alias — one command to pipe
  • 10 MB paste limit
  • No JavaScript required
  • IP privacy (SHA-256 hashed)
